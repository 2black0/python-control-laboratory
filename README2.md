# Jobsheet Praktikum Sistem Kendali dengan Python dan GNU Octave

Ini adalah Jobsheet untuk Praktikum Sistem Kendali dengan Python dan GNU Octave untuk mahasiswa D4 Teknik Elektronika Fakultas Vokasi UNY.

## Description

Praktikum Sistem Kendali dengan Python dan GNU Octave adalah salah satu praktikum yang ditawarkan dalam program studi yang berkaitan dengan rekayasa kontrol atau bidang terkait. Praktikum ini bertujuan untuk memberikan pemahaman dan pengalaman praktis dalam menerapkan konsep sistem kendali menggunakan dua bahasa pemrograman populer, yaitu Python dan GNU Octave.

Dalam praktikum ini, mahasiswa akan belajar mengenai prinsip dasar sistem kendali dan metode-metode yang digunakan untuk menganalisis dan merancang sistem kendali. Praktikum ini menggabungkan teori dengan penerapan langsung menggunakan perangkat lunak open-source, seperti Python dan GNU Octave, yang memberikan fleksibilitas dan kemudahan dalam memahami konsep-konsep yang diajarkan.

Pada awal praktikum, mahasiswa akan diperkenalkan dengan dasar-dasar sistem kendali, termasuk konsep umpan balik, fungsi transfer, diagram blok, dan metode analisis frekuensi. Kemudian, mahasiswa akan belajar menggunakan bahasa pemrograman Python dan GNU Octave untuk melakukan simulasi sistem kendali, menganalisis respons sistem, dan merancang kontroler yang sesuai.

Selama praktikum, mahasiswa akan diberikan serangkaian tugas praktis, di mana mereka akan diberikan permasalahan kontrol yang berbeda dan diminta untuk mengimplementasikan solusi menggunakan Python dan GNU Octave. Mahasiswa akan belajar menggunakan library atau paket yang relevan, seperti NumPy, SciPy, Control, dan Octave Control Systems Toolbox, untuk memudahkan analisis dan desain sistem kendali.

Beberapa topik yang akan dibahas dalam praktikum ini meliputi:

Analisis respons sistem kendali menggunakan Python dan GNU Octave.
Merancang kontroler PID (Proporsional-Integral-Derivative) menggunakan metode tuning.
Merancang kontroler State-Space menggunakan pendekatan pemrograman linier.
Merancang kontroler menggunakan metode optimal seperti Linear Quadratic Regulator (LQR) dan H-infinity.
Simulasi sistem kendali dalam berbagai kondisi dan lingkungan menggunakan Python dan GNU Octave.
Praktikum Sistem Kendali dengan Python dan GNU Octave ini diharapkan dapat membantu mahasiswa dalam memahami dan menguasai konsep sistem kendali serta penerapannya menggunakan bahasa pemrograman yang umum digunakan di industri. Praktikum ini juga memberikan dasar yang kuat bagi mahasiswa yang berminat melanjutkan studi atau karir di bidang rekayasa kontrol dan automasi.
